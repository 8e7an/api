# Image

## Image.width

    var w = image.width

## Image.height

    var h = image.height

## Image.putPixel()

    image.putPixel(x, y, color)

Sets the pixel in the *xy*-coordinate to the given
[pixel color](pixelcolor.md).

## Image.getPixel()

    var color = image.getPixel(x, y)

Returns the [pixel color](pixelcolor.md) for the given *xy*-coordinate.
